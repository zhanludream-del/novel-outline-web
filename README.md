# 晨曦微露AI创作软件 Web版

## 🎯 特点

✅ **纯前端实现** - 无需服务器，双击`index.html`即可打开  
✅ **完全免费** - 数据存储在浏览器本地，不产生任何费用  
✅ **移动端适配** - 支持手机、平板、电脑浏览器访问  
✅ **跨平台** - Windows/Mac/Linux/iOS/Android都可以使用  
✅ **数据安全** - 所有数据保存在本地，不上传到任何服务器  

## 🚀 快速开始

### 方式1：本地打开（推荐）
1. 双击 `index.html` 文件
2. 在浏览器中打开
3. 点击右上角"⚙️ 设置"配置API Key
4. 开始使用！

### 方式2：GitHub Pages部署（手机访问）
1. 将整个文件夹上传到GitHub仓库
2. 开启GitHub Pages
3. 用手机浏览器访问你的GitHub Pages地址

## ⚙️ API配置

### 豆包API（推荐）
1. 访问 [火山引擎](https://www.volcengine.com/)
2. 注册并开通豆包API
3. 获取API Key
4. 在设置中填入API Key

### OpenAI API
1. 选择提供商为"OpenAI"
2. 填入OpenAI API Key
3. 如需代理，修改API Base URL

## 💾 数据存储

- **自动保存**：数据自动保存在浏览器LocalStorage中
- **导出备份**：点击"📤 导出"下载JSON文件
- **导入恢复**：点击"📥 导入"加载JSON文件
- **跨设备迁移**：导出JSON文件后，在其他设备导入即可

## 📱 移动端使用

1. **手机浏览器访问**：部署到GitHub Pages后，用手机浏览器打开
2. **适配优化**：
   - 侧边栏自动隐藏，点击左上角☰按钮展开
   - 按钮自动调整为竖排布局
   - 字体大小适配手机屏幕

## ❓ 常见问题

### Q: API Key安全吗？
A: API Key只存储在浏览器本地，不会上传到任何服务器。但请注意不要在公共电脑上保存API Key。

### Q: 数据会丢失吗？
A: 数据保存在浏览器LocalStorage中，清除浏览器数据会导致数据丢失。建议定期导出JSON文件备份。

### Q: 支持哪些浏览器？
A: 推荐使用Chrome、Edge、Firefox、Safari等现代浏览器。不支持IE浏览器。

### Q: 手机上如何使用？
A: 部署到GitHub Pages或其他静态托管服务后，用手机浏览器访问即可。

### Q: 能否同时使用多个API？
A: 目前只能配置一个API，但可以在设置中随时切换。

## 💡 使用建议

1. **首次使用**：先配置API Key，然后生成大纲
2. **定期备份**：每次重要修改后点击"导出"保存JSON文件
3. **手机使用**：建议横屏使用，体验更好
4. **API选择**：豆包API性价比高，OpenAI效果更好
