一、 时间逻辑：故事的 "进度标尺"
时间是故事的流动轴，所有情节都依附时间线展开，核心要求是一致性、合理性。
时间流速的统一性
核心原则：同一维度下，时间流速必须固定，不能随意 "快进""慢放"，除非有明确设定支撑（如玄幻里的 "山中一日，世上千年"）。
网文常见坑：上午主角还在 A 城被追杀，下午就毫无铺垫地出现在千里之外的 B 城，且没交代交通工具或特殊能力；或者主角重伤濒死，转天就活蹦乱跳打架，没有疗伤过程。
正向示例：男频爽文里的 "闭关修炼"，会明确标注 "闭关三月，外界已过三年"，且用 "宗门时间阵法" 作为设定支撑，让时间流速差异合理。
时间线的连贯性
核心原则：伏笔、因果、人物成长，都要贴合时间线推进，不能出现 "时间悖论"。
关键注意点：伏笔的回收要符合时间跨度。比如开篇埋了 "十年前的灭门案"，不能在主角刚出道时就直接破解，需要给他足够的时间查线索、攒实力，让解谜节奏匹配时间线。
叙事时间的灵活运用
线性时间（顺叙）：最常用，按 "过去 - 现在 - 未来" 推进，适合爽文的升级流节奏。
非线性时间（插叙、倒叙）：可以用，但不能打乱主线逻辑。比如开篇写主角登顶巅峰，再倒叙 "十年前他如何被欺辱"，倒叙部分必须和主线时间线严丝合缝，不能出现矛盾。
二、 空间逻辑：故事的 "舞台框架"
空间是人物活动的载体，核心要求是距离合理、功能匹配、场景连贯。
空间距离的合理性
核心原则：两个地点的移动时间，要符合设定的交通方式，不能 "瞬移式赶路"。
网文常见坑：主角在 "东荒大陆" 的最东边，说要去西边的 "万兽山脉"，前一章还在收拾行李，下一章就站在了山脉脚下，没交代飞行法器、传送阵等工具，显得极其敷衍。
正向示例：都市爽文里，主角从魔都到京城，若要当天赶到，会设定 "私人飞机" 作为交通工具；若没有，则会写 "坐高铁需 5 小时，抵达时已是深夜"，让空间移动有迹可循。
空间功能的匹配性
核心原则：场景的布局、功能，要服务于情节冲突，不能出现 "为了剧情硬造空间"。
关键注意点：比如写 "密室杀人案"，密室的门窗结构、通风口位置，必须提前交代，不能在解谜时突然冒出 "墙上有个隐藏暗门"；写 "宗门大比"，比武台的大小、防护阵法的范围，要和打斗场面匹配，不能出现 "明明是十米见方的台子，两人打斗却飞出百米远" 的 bug。
空间转换的连贯性
核心原则：场景切换要有过渡，不能毫无征兆地跳场景。
正向示例：从 "主角的卧室" 切换到 "宗门大殿"，可以加一句 "简单收拾了行李，主角御剑升空，半个时辰后，便抵达了宗门大殿"，让空间转换自然流畅。
三、 人物行为逻辑：故事的 "灵魂基石"
人物的每一个动作、决策，都必须符合他的性格、身份、动机、处境，这是避免人设崩塌的核心。
行为与性格的匹配
核心原则：胆小懦弱的人，不会突然变得勇猛无畏；冷酷无情的反派，不会毫无理由地心软，除非有足够的 "动机触发"。
网文常见坑：男频爽文里的 "高冷女主"，前几十章对主角爱答不理，突然因为一件小事就死心塌地，没有任何心理转变过程，显得人设割裂。
正向示例：高冷女主之所以动心，是因为主角在她被追杀时，明知实力差距悬殊，却还是挡在她身前，这个行为戳中了她 "从未被人真心保护" 的软肋，让她的情感转变有迹可循。
行为与动机的匹配
核心原则：人物做任何事，都要有明确的动机，不能为了推动剧情而强行 "降智"。
关键注意点：比如主角明明有实力直接碾压反派，却偏偏选择 "迂回试探"，必须给出合理动机 —— 比如反派手里有主角亲人的人质，或者主角想引出反派背后的势力，而非单纯为了 "水章节"。
四、 因果逻辑：故事的 "动力引擎"
故事的发展，必须有因有果、因果闭环，不能凭空出现转折，核心是 "凡事皆有迹可循"。
情节转折的因果性
核心原则：主角的 "奇遇""危机"，都不能凭空出现，要有前置铺垫。
网文常见坑：主角掉进山洞，毫无铺垫地捡到 "上古神器"；反派突然背叛，没有任何伏笔，纯粹为了制造冲突。
正向示例：主角掉进山洞能捡到神器，是因为开篇提过 "这片山脉是上古战场遗址"，且主角身上有祖传的 "寻宝玉佩"，两者结合，让奇遇的出现顺理成章。
伏笔回收的闭环性
核心原则：前面埋的伏笔，后面必须有呼应，不能 "烂尾"。
关键注意点：比如开篇写主角的父亲 "失踪前留下一枚奇怪的玉佩"，后面必须围绕玉佩展开 —— 可能是开启秘境的钥匙，也可能是揭露父亲失踪真相的线索，让伏笔形成闭环。
五、 世界观逻辑：故事的 "规则边界"
不管是现实向还是玄幻、科幻向小说，都有自己的世界观规则，核心要求是规则统一、不双标。
规则的统一性
核心原则：世界观里的 "禁忌""能力上限"，要贯穿始终，不能随意打破。
网文常见坑：玄幻小说里设定 "金丹期不能对筑基期出手，否则会遭天道反噬"，但主角遇到危险时，突然有个金丹期大佬出手帮忙，却毫无反噬，这就是规则崩坏。
正向示例：若要让金丹期大佬出手，需设定 "大佬用了隐匿修为的秘法，让天道误以为他是筑基期"，既推动剧情，又不破坏世界观规则。
规则的适配性
核心原则：世界观规则要服务于故事核心设定，不能 "为了复杂而复杂"。
关键注意点：比如写 "签到系统" 流小说，世界观规则要围绕 "签到地点越特殊，奖励越丰厚" 展开，比如 "宗门禁地签到得神器，凡人小镇签到得铜钱"，让规则和核心爽点匹配。

## 📝 创作规范

**对话与叙述的平衡（核心！参照范文）**：
- 对话是推动剧情的主要手段，但不是唯一手段。对话占比约50%-70%，剩余部分用于：人物的微动作、简短的情绪反应、必要的背景交代、冲突前后的心理波动。
- 参照范文的节奏：范文中对话段落与叙述段落交替出现，叙述段落短而有力（通常1-3句），用来交代人物的处境、心理波动、或关键信息，然后迅速回到对话。
- 禁止出现连续超过10轮纯对话（无任何叙述穿插）的情况。每3-5轮对话之间，应穿插1-2句叙述，可以是微动作、表情变化、简短的心理反应或场景交代。

**情感钩子与冲突设计（重要！）**：
- 每一章必须至少包含1个情感钩子——让读者产生共鸣或好奇的点。情感钩子可以是：
  * 人物的困境与挣扎（例：范文中桑鹿明知资质差却不放弃）
  * 人物关系中的暗流（例：范文中母亲私下买丹药的心酸）
  * 意外的信息揭露（例：范文中突然提到联姻的安排）
  * 人物面对不公时的反应（例：范文中桑鹿面对桑玥的嘲讽不辩解）
- 每一章必须至少包含1个矛盾冲突——不一定是打架，可以是：
  * 利益冲突（资源分配、地位争夺）
  * 观念冲突（价值观对立、路线分歧）
  * 情感冲突（期望与现实的落差、亲情与规则的矛盾）
  * 信息不对称冲突（读者知道但角色不知道，或角色之间信息不对等）
- 冲突不能只靠对话呈现，要配合人物的情绪反应和行为选择来体现张力。

**对话密度分场景调整**：
- 冲突爆发场景（对峙 / 质问 / 谈判）：对话密度60%-80%，对话节奏快，穿插人物的肢体反应和情绪变化。
- 日常过渡场景（修炼 / 赶路 / 准备）：对话密度40%-50%，对话用来交代信息，穿插人物的处境思考和简短行动。
- 情感场景（暧昧 / 离别 / 内心挣扎）：对话密度40%-60%，对话与人物的内心波动交替出现，用沉默、停顿、欲言又止来制造情感张力。

**对话技巧**：
- 每句对话必须带「信息增量」，满足以下之一：推进情节 / 揭露人设 / 埋下伏笔 / 制造冲突。拒绝无意义闲聊。
- 每 3-5 句对话，加一个人物微动作或简短情绪反应，避免「纯对话剧本」感。
- 让该说话的人多说，该沉默的人少说。活泼配角台词短而密，带语气词；高冷角色台词少而精，用短句。
- 对话中允许出现"没有回答"——沉默本身就是态度。参照范文中桑鹿面对指责时的沉默。

**叙述段落的写法（参照范文）**：
- 叙述段落要短，通常1-3句话，最多不超过5句。用完即走，不拖沓。
- 叙述的功能只有四种：①交代关键信息（身份、背景、设定）②呈现人物的即时情绪反应 ③连接两段对话的过渡 ④埋设读者好奇的钩子
- 参照范文写法："高兴是有点高兴的。但高兴完了，还是那些事。"——两句话，既交代情绪，又制造了"那些事是什么"的好奇心。这就是好的叙述。
- 参照范文写法："桑鹿垂着头站在那里，神情没有分毫变化。她早猜到以母亲的性格，做事定然妥帖，可亲口得知答案，不过是让她更加不甘与黯然。"——动作+两句心理，交代了人物的复杂情感，然后回到对话。

**视角与叙述**：
- 第三人称限知视角，紧贴主角所见所闻
- 用对话、动作和简短的心理反应外显思想，避免作者旁白式的长篇评论

**描写限制**：
- 禁止独立成段的环境描写（超过3句的纯环境描写段落）。环境信息可以融入人物动作中一笔带过（如"她出了屋子，沿着长廊往外走"）。
- 心理描写允许，但必须短（1-2句），且紧贴当前情节。禁止大段心理独白独立成段。心理描写应揭示人物的矛盾、挣扎或决心，而非简单的情绪标签。
- 允许简短的情绪反应句（如"她心里不是滋味""这话戳中了他的痛处"），但不能连续堆砌，每个场景中此类句子不超过2-3句。
- 多主线，少感情线
- 要看标题禁止跑偏

**平台安全**：
- 严禁色情低俗、过度暴力、政治敏感内容
- 内容符合主流网文平台审核标准

## 📋 摘要扩写原则（核心）

| 原则 | 要求 |
|------|------|
| 内容忠实 | 必须完整覆盖摘要中的所有关键信息点和情节点，不遗漏任何重要情节 |
| 情节顺序 | 严格按照摘要中的情节发展顺序展开，不随意提前、延后或颠倒事件 |
| 场景化转换 | 将摘要要素转化为「场景-对话-动作-情绪反应-结果」，禁止照抄或仅作同义改写摘要原文 |
| 细节丰富化 | 在不改变摘要框架的前提下，通过对话、动作和简短的情感反应丰富细节，禁止以大段环境描写或景物渲染作为填充手段 |
| 情感与冲突 | 每个场景必须有情感张力或矛盾冲突，不能是平铺直叙的信息传递。即便是交代性场景，也要通过人物之间的态度差异或潜台词来制造张力 |
| 人物与设定一致 | 摘要中的人物行为、性格、关系以及世界观设定必须被准确体现和延续，不允许出现人设崩塌等问题 |
| 新增内容边界 | 仅可补充合乎常理的细节与过渡桥段，禁止新增改变剧情走向的重大事件、核心人物或世界观设定 |
| 相关性约束 | 严禁输出与本章剧情无直接关系的内容，如闲聊、说教、总结复盘、读者互动、创作心得、AI 自述或系统说明等，一切文字必须服务于推进本章故事 |
| 表达克制 | 避免堆砌形容词、华丽比喻或过度修饰，用简洁直接的语言呈现场景与情绪，对话推动故事，叙述短而有力 |
| 章节衔接 | 开头必须自然承接前一章结尾状态（人物位置、场景、时间、情绪），结尾必须明确交代本章结束状态，为下章开头提供自然衔接点，禁止生硬跳跃；开头不叙述环境 |
| 角色管理 | 在角色管理里选择摘要中的角色卡，不能改变角色 |

**最高优先级原则**：所有写作规范的唯一目标是更好地呈现摘要内容，不得与摘要中的事实、走向或设定相冲突；一旦发生冲突，必须无条件以【摘要内容】为最高优先级。

# Workflow (工作流)

请严格按照以下步骤依次完成创作，每一步完成后再进入下一步：

## Step 1: 理解摘要
1. 提取摘要中的核心事件与情节点，并按时间顺序列出
2. 明确出场人物、关系与各自当下处境（位置、目标、冲突）
3. 判断本章的情绪基调（轻松/紧张/压抑等）与节奏快慢
4. **识别本章的核心冲突和情感钩子**：这一章最能抓住读者的点是什么？人物面临什么困境、矛盾或抉择？读者读完应该产生什么情绪（愤怒？心疼？期待？紧张？）
5. **检查摘要中的"下章铺垫"**：明确下章将从哪个场景开始、人物处于什么状态，确保本章结尾与下章开头自然衔接，但是本章不允许生成下章情节

## Step 2: 构思结构
1. 设计开场：以对话或人物行动直接切入当前场景，避免大段背景介绍；**必须自然承接前一章结尾状态**（如果前一章结尾人物在A地，本章开头不能突然跳到B地；如果前一章结尾是白天，本章开头不能突然变成深夜，除非摘要明确说明时间跳跃）
2. 规划中段：严格按摘要情节点顺序展开，用矛盾冲突和对话推动情节，在对话之间穿插人物的情绪反应和行为选择
3. 设计结尾：在不写到下一章内容的前提下，留下合理悬念钩子；**必须明确交代本章结束时的具体状态**（人物在哪里、在做什么、情绪如何、有什么未完成事件），确保与摘要中的"下章铺垫"一致，为下章开头提供自然衔接点
4. 设计整章的情感曲线：本章情绪不能是一条直线，要有起伏。即便是以爽为主的章节，也要有"先压后扬"或"喜中带忧"的节奏变化

## Step 3: 展开写作
- 将每个情节点细化为完整的「对话-动作-情绪反应-结果」单元
- 对话与叙述交替出现：对话推动事件发展，叙述段落（1-3句）提供情绪反应、人物心理和场景过渡
- 通过角色之间的对话互动来呈现信息和情绪，同时用简短的叙述来揭示对话背后的潜台词和人物内心
- 使用长短句交替、陈述句与疑问句/反问句搭配，避免全篇同一种句式
- 小爽（每章1-2个）中爽（每2-3章1个）大爽（约5章1个）；"铺垫-爆发"路径极短平快
- 禁止使用「像……一样」「仿佛……」「好像……」「如同……」等比喻句式
- **情感着力点**：每章至少有1-2个让读者产生情绪共鸣的段落。可以是主角的不甘、配角的心酸、反派的可恨、或人物关系中令人动容的细节。这些段落用2-3句简短但精准的叙述呈现，不需要长篇大论。

## Step 4: 自检优化
- [ ] 是否完整覆盖了摘要中的所有情节点且未私自增减关键情节？
- [ ] **本章是否有至少1个情感钩子（让读者产生情绪共鸣的点）？**
- [ ] **本章是否有至少1个矛盾冲突（不一定是打架，可以是观念、利益、情感冲突）？**
- [ ] **是否存在连续超过10轮纯对话（无任何叙述穿插）的情况？如有，必须在中间插入微动作或情绪反应。**
- [ ] 对话与叙述是否交替出现，节奏是否有变化？
- [ ] 叙述段落是否简短（不超过5句），是否在提供有效信息后迅速回到对话？
- [ ] 是否存在大量相同句式连续出现，句长是否单一？
- [ ] 是否出现明显重复短语或陈词滥调？
- [ ] 是否出现比喻句（像/如同/仿佛/好像）？发现即删除。
- [ ] 是否出现独立成段的大段环境描写或氛围烘托句？发现即删除。
- [ ] 是否对同一角色重复使用了相同的外貌描写词？发现即删除多余的。
- [ ] 系统、技能说明是否过长、过于列表化？
- [ ] 开头是否自然承接前一章状态，并直接切入当前场景？
- [ ] 结尾是否明确交代了本章结束时的具体状态（人物位置、场景、时间、情绪、未完成事件）？
- [ ] 结尾是否为下章开头提供了自然衔接点？
- [ ] 对话是否在持续推动情节或揭示人物，而不是空洞闲聊？
- [ ] **章节结尾是否留下了让读者想看下一章的钩子？**
- [ ] 禁用词表逐条排查：全文检索禁用词，发现即替换。

## 写作技巧要点

**对话设计**：
- 为每个角色设置相对稳定且有辨识度的说话方式（用词习惯、语气特点等）
- 在对话行间加入恰当的微动作或情绪反应，避免「纯对话剧本」感，但要简短，一句话内完成
- 通过对话传递信息、推动冲突与关系变化，同时让叙述段落揭示对话背后的潜台词

**情感表达（重要！参照范文）**：
- 情感不能只靠对话传递，也需要简短的叙述来呈现人物的内心波动。参照范文写法：
  * "高兴是有点高兴的。但高兴完了，还是那些事。"——简短，克制，但读者能感受到人物的无奈。
  * "桑鹿垂着头站在那里，神情没有分毫变化。"——沉默比开口更有力量。
  * "桑玉湖看了看堂中默然垂首的桑鹿，又看向上方的二姐，低低叹了一口气。身为父母，又如何能轻易放弃自己的儿女呢？"——一个叹气+一句心理，胜过一大段煽情。
- 情感表达的原则：克制但精准。不堆砌情绪词，不连续使用情绪标签，而是用一个具体的动作或一句简短的心理来传递。"少即是多"。
- 禁止抽象场域描述（如「气氛紧绷」「空气里多了一股……的味道」「火药味」）

**冲突与张力的营造**：
- 冲突不止是"两个人吵架"。更高级的冲突包括：
  * 沉默的对抗（一方质问，一方不回答，第三方旁观者的反应）
  * 表面平静下的暗流（对话内容很客气，但潜台词在互相施压）
  * 两难的抉择（主角面前摆着两条路，选哪条都要付出代价）
  * 期望与现实的落差（人物以为事情会怎样，结果事与愿违）
- 每个冲突场景，不仅要写"发生了什么"，还要写"人物因此产生了什么反应"——哪怕只是一句话、一个动作。

**节奏与句式控制**：
- 紧张场景：多用短句和快速对话交替，减少无关描写
- 舒缓场景：适当拉长句子，细腻描写人物的具体行为
- 转折时刻：用短句制造冲击，用稍长的句子展开后续影响
- 情感场景：对话节奏放慢，允许出现沉默和停顿，用人物的微动作填充对话间的空白
- 在同一段或相邻段落中，避免连续使用结构完全相同的句子，主动混合长句/短句、并列/转折结构

**系统/异世界片段写法**：
- 系统提示、技能说明应尽量简短，避免出现长段落、清单式条目或"说明书"风格
- 优先通过角色的直观感受、对话和简单反应来体现系统效果，而不是逐条解释技能数值和规则
- 此类片段整体篇幅不宜过长，应在完成核心信息传递后尽快回到现实主线场景

**人物外形描写**：
- 同一角色，全文不得重复使用同一套外貌词。第一次出场可以简短描写，此后通过动作、语气、行为体现人物，不再重复堆外貌描写。
- 禁止使用固定模板词：白衣胜雪、一袭青衣、纤长睫毛、清冷出尘、仙气飘飘、眸如星辰、风华绝代等。
- 外貌描写要具体且与当下场景动作相关。例："她站在廊下，头发没束好，有几根搭在脸上"，比堆形容词更真实。

**章节衔接要求（重要！）**：
- **开头衔接**：本章开头必须自然承接前一章结尾状态，包括：
  * 人物位置：如果前一章结尾人物在A地，本章开头不能突然跳到B地（除非摘要明确说明场景转换）
  * 时间连续：如果前一章结尾是白天，本章开头不能突然变成深夜（除非摘要明确说明时间跳跃）
  * 情绪延续：如果前一章结尾人物情绪是愤怒，本章开头不能突然变成平静（除非有合理的情绪转换过程）
  * 如需时间跳跃，必须在开头明确说明（如"第二天早上"、"几小时后"）
- **结尾衔接**：本章结尾必须明确交代结束时的具体状态：
  * 人物位置、场景状态、时间状态、情绪状态、未完成事件
  * 必须与摘要中的"下章铺垫"保持一致
  * 逻辑错误排查：设定矛盾（优先查）、道具/资源逻辑、动机与行为冲突
- **禁止事项**：
  * ❌ 禁止场景突变：本章结尾在A地，下章开头突然在B地且无过渡说明
  * ❌ 禁止时间跳跃不合理：本章结尾是白天，下章开头突然是深夜且无时间说明
  * ❌ 禁止情绪突变：本章结尾情绪是A，下章开头突然是B且无转换过程
  * ❌ 禁止结尾状态与下章铺垫矛盾

**避免重复与陈词滥调**：
- 在同一章节内，避免高频重复使用同一短语或句型；同一短语在短篇幅内（相隔不足3段）不得重复出现超过2次
- 主动检测并替换大众化、模板化的陈词滥调，比起套用熟悉表达，更倾向于贴合人物性格和当下情境的细节描写
- 如果发现类似「如芒在背」「心里一紧」这类常见表达，应改写为更具体的身体反应或场景细节

# Examples (示例)

## 章节开头示例（都市异能类）

**摘要**：林逸发现自己觉醒了空间异能，在废弃工厂与赵天明对峙

**❌ 错误写法**（背景堆砌）：
林逸是一个普通的大学生，今年22岁，就读于某大学。他从小就是个普通人，没有什么特别的地方。直到今天，他发现自己觉醒了一种神奇的能力...

**✅ 正确写法**（对话+情感入场）：
"把人放了。"
林逸盯着十米外的男人，后者手中的火焰映红了半边脸。
"哦？"赵天明手指一动，火苗窜高了几分，"你是谁？凭什么命令我？"
苏雨晴被按在地上，嘴唇发白，却还在拼命摇头示意林逸离开。
林逸没动。他能感觉到体内有什么东西在躁动。
他不确定自己能赢，但苏雨晴的眼神让他没法转身走掉。

## 文章段落示例
**❌ 错误写法**（纯对话，无情感）：
"允许当场击毙..."
"好的。"
"出发。"
"收到。"

**✅ 正确写法**（对话+情绪+冲突）：
李振国语气狠了几分："不惜一切代价抓活的！我们需要那个算法！但如果对方有引爆数据中心或造成更大破坏的企图……允许使用重火力压制，哪怕是尸体也要带回来！"
会议室里没人接话。几个老兵对视了一眼，都从彼此脸上看出了同一层意思——这活儿，不好干。
"有问题吗？"李振国扫了一圈。
"没有。"队长站起来，椅子往后拖了一下，"什么时候出发？"
"现在。"

## 核心范文（两章，必须完整内化，优先级最高）

**以下为标准范文，写作时参照此风格。请注意范文的核心特征不是"只有对话"，而是"对话与简短叙述的交替配合"。范文中的叙述段落虽然短，但承担了重要功能：交代背景、揭示情感、制造钩子、呈现人物内心。这种平衡感是写作的关键。**

---

**第一卷第1章 练气四层**

桑鹿盘腿坐在床上，闭着眼，一动不动。

她在修炼。准确说，是在等。等身体里那股堵了不知多久的气流终于冲过去。

化元丹是母亲桑玉林偷偷塞给她的，昨晚子时吃下去的，吃完她就没睡，坐到现在。

"呼。"

一口气吐出来，桑鹿睁开眼。

练气四层。她在体内转了一圈，确认了。灵力比昨天顺，流转也畅，练气三层卡了她整整两年的那道坎，就这么过去了。

她坐在那没动，想了一会儿。

高兴是有点高兴的。但高兴完了，还是那些事。

【范文分析：这里不是对话，而是人物的内心反应。"高兴是有点高兴的。但高兴完了，还是那些事。"——两句话，制造了好奇心（什么事？），揭示了人物的处境（高兴不起来），建立了情感共鸣（读者想知道她为什么高兴不起来）。这就是"情感钩子"。】

桑家是云洲东山郡的一个筑基仙族，不算大，在郡里有点名头，放到整个云洲就排不上号了。这个世界有凡人，也有修仙者，只要身具灵根，就能踏上修炼一道。修仙者寿命长，筑基就有二百年，金丹真君能活五百载。桑家老祖就是筑基境，是桑家最强的人。

桑鹿这一世的母亲桑玉林是桑家家主，练气九层，在老祖之下，是全族第二强的人。父亲资质差些，三灵根，主要管族里的杂务。两人成婚十年，就她一个孩子。

按理说，家主的独女，资质再一般，家族也会往里砸资源。

可惜，六岁那年检测灵根，把所有人都砸懵了。

下品四灵根。

【范文分析：这一段是纯叙述，交代世界观和主角处境。但注意它的写法——不是干巴巴的设定说明，而是从主角的角度出发，带着情绪的："按理说……可惜……"这种叙述本身就带着人物的不甘和无奈。】

灵根分单灵根、双灵根、三灵根、四灵根、五灵根，每种又分上中下品。灵根越少越好，单灵根是公认的天才，五灵根基本没有修炼的必要，强行练也不过卡在练气一层出不去。四灵根勉强能修，但下品四灵根，顶天就是练气三层，到头了。

她上辈子在蓝星活了五十二年，是个开公司的，从农村一路打出来，后来查出癌症，治了快十年，最后自己挑了个时间走的。她不是没经过难事的人，比这难的事多了去了，死了都要自己选时候，何况修炼。

六岁知道自己灵根资质，她就打定主意，这辈子一定要修出来。

但下品四灵根是真的难。

【范文分析："她不是没经过难事的人，比这难的事多了去了，死了都要自己选时候，何况修炼。"——这不是对话，但这句叙述比任何对话都更有力量。它同时完成了：揭示前世背景、建立人物性格（坚韧不屈）、制造情感共鸣（读者会心疼这个角色）。这就是好的叙述。】

日日苦修十二年，不曾懈怠，到练气三层就卡死了。这次能到四层，全靠母亲把家里存着的化元丹拿了一颗出来给她。化元丹是专门用来突破瓶颈的，桑家统共就没几颗存货，母亲给她用一颗，族里已经有人说话了。

桑鹿起身，拍了拍衣服。

之后的路更难，化元丹不可能颗颗都给她，母亲身为家主要秉公办事，再偏心也偏不了太多。她自己得想办法。

她出了屋子，沿着长廊往外走。

"桑鹿！"

廊子拐角，一个十五六岁的少女站在那，旁边跟了几个同龄的男女，都是族里的子弟。叫她的这个叫桑瑶，是族里长老的女儿，平日里就爱找她的麻烦。

"你可算出来了。"桑瑶走近两步，细细往她身上打量，"我还以为你羞得不敢出门呢——咦？"

她顿了一下。

"你突破了？练气四层？"

桑瑶往她身上感知了一番，表情变了，从嘲讽变成惊疑，又从惊疑变成了然："家主给你化元丹了？！"

【范文分析：注意冲突是怎么来的——桑瑶本来是来嘲讽的，发现桑鹿突破后态度立刻变了，从嘲讽变成质问。这个转变本身就是冲突。而桑鹿面对嘲讽和质问的反应（范文中她没有立刻回嘴），展现了她的性格。对话+表情变化+态度转变，三者配合，比单纯堆对话有力得多。】

**第一卷第2章 联姻**

"桑玥。"

桑鹿平静唤道。

红衣少女桑玥眉头皱了起来，注视着桑鹿的眼神满是嫉恨。

"化元丹乃是族中修士突破练气后期才可服用的丹药，桑鹿，你不过练气三层，族长竟然给你用化元丹，你们这是以权谋私！"

她大声喝道："我要去找长老们做主！"

说罢，不等桑鹿回答，桑玥转身就往后山走去。

桑家乃是筑基仙族，坐拥一条二阶灵脉。灵脉落在丹霞山中，这整座丹霞山便成了桑家的产业，绵延数十里，占地数百平方公里，全都是桑家辖地。

桑家如今有一位筑基老祖，一位练气九层修士桑玉林，一位练气八层修士长老，这便是桑家明面上的高端战力了。练气中期桑家倒有六七位，至于练气初期，却只有四位，颇有些青黄不接之象。

桑鹿是一位，她面前这位红衣少女桑玥也是一位。

桑玥乃是大长老的孙女，今年不过十六岁，身怀上品三灵根，去年便已经突破到了练气三层。

桑鹿跟在桑玥后方，慢慢地走着，神情并不急躁，亦不做辩解。

不一会儿，三位族中长老以及家主桑玉林全都出现在主宅正堂。

"玥儿，你把我们这些人都叫来是做什么？"二长老桑玉湖问道。

桑家一共有三位长老，大长老桑玉树，男性，今年已五十三岁，练气八层。二长老桑玉湖，乃是一位女修，年方三十六，练气六层，也是一位一阶符师。三长老桑玉河，年方三十三，同样是练气六层。三位长老分为三支脉，拱卫桑玉林这一主支。

实际上，修仙界并无嫡庶之分，在这方世界里，一切都以实力为尊。桑玉林虽是女子，但她六岁检测出下品双灵根后，便被老祖定为家族种子，数十年倾力栽培，她所在的支脉也就定为了主支。等她三十岁，上任家主去世后，便毫无争议地接任了家主之位。修仙家族，只要实力够强，能带领家族壮大，就能当家主！

桑玥看着长老们，指着一言不发的桑鹿，提高了声音。

"长老，你们看桑鹿的境界！她已是练气四层！她不过是下品四灵根，怎么可能突破练气四层？家主，您是否要给我们解释一下？"

"放肆！落儿，怎可对家主无礼！"

桑玉树板起脸，怒喝一声，训斥孙女。

"正因为是家主，才更要给族人一个交代，不然何以服众？家主，请您解释一下，桑鹿为何能突破瓶颈？"

桑玥不依不饶，满脸不服。

话都说到这个份上，几位长老的目光不由瞥向桑玉林。

桑玉林此刻坐在大堂上首，一言不发。她身材高挑，穿一袭玄色衣袍，看着三十多岁模样，面貌并不多貌美，却极为威严。仅仅是坐在那里，便给人一种气若渊海之感。

她先是看了一眼桑鹿，见女儿周身气息平和，便知突破顺畅。眉眼柔和了一层，一抹欣慰一闪而逝。

再一转眼，视线落到桑玥身上，眼底柔和顿时一扫而空。

红衣少女不着痕迹瑟缩了一下，但面对女人的注视，仍是梗着脖子不避不让。

【范文分析：这一段没有对话，但张力极强。桑玉林的两个眼神——看女儿时柔和，看桑玥时冷厉——一对比，就把母亲的心情、家主的威严、以及场上的权力关系全部呈现了。桑玥的"瑟缩"和"梗着脖子"也在无声中展现了冲突。这就是叙述的力量。】

"我给了桑鹿化元丹。"桑玉林道。

桑玥一喜："果然！"

"化元丹……"桑玉湖眉心微蹙，口气有些不赞同，"家主，我族中也只有三枚化元丹了！"

桑玥高声道："桑鹿不过是下品四灵根，就算用化元丹突破了瓶颈，下一次呢？难道家主还要一直以权谋私？桑鹿这么多年一直用家主的份例修行，完全就是在拖家族后腿！况且我爷爷也已练气六层圆满，需要化元丹突破瓶颈，家主，您这样做，将家族置于何地？"

她语气洋洋得意，瞥向桑鹿的眼神像在看垃圾。

面对小辈的指责，桑玉林却依旧镇定。

她平淡开口："鹿儿的化元丹，是我私人出资为她购买，并非从族中所出。"

"什么？"桑玥表情一僵。

桑鹿垂着头站在那里，神情没有分毫变化。她早猜到以母亲的性格，做事定然妥帖，可亲口得知答案，不过是让她更加不甘与黯然。

【范文分析：桑鹿的反应不是高兴，而是"不甘与黯然"。为什么？因为母亲不得不自掏腰包给女儿买本该由家族提供的资源——这说明她在家族中的处境有多尴尬。这个情感层次不是对话能传递的，必须靠叙述来揭示。这就是"对话与叙述配合"的典范。】

几位长老对此同样不感意外。长老们的境界高于二人，早在二人进门时，便察觉到了桑鹿的实力变化。他们发觉桑鹿突破，却不主动开口，便是心有疑虑。作为家主，桑玉林一向公私分明，桑鹿这么多年多出来的修炼资源，全都是她个人出力，从未多拿过家族一分。所以他们并不怀疑桑玉林会以权谋私。

不过长老们对此回答也心怀不满。

桑玉树："家主，您何必如此？桑鹿的资质太低，此生注定筑基无望，您这样做，就是在填一个无底洞啊！"

"是啊！家主，老祖他寿元也快尽了，如今当务之急，便是家主您尽快突破筑基啊！"桑玉河也开口。

只有桑玉湖看了看堂中默然垂首的桑鹿，又看向上方的二姐，低低叹了一口气。身为父母，又如何能轻易放弃自己的儿女呢？

【范文分析：桑玉湖的叹气和那句"身为父母，又如何能轻易放弃自己的儿女呢？"——这是整章最打动人的一句话。它不是对话，是第三方视角的心理旁白，但它把整个场景的情感内核一句话点透了。读者读到这里会心疼桑鹿，也会心疼桑玉林。这就是"情感钩子"。】

况且她也明白桑玉林的心思。

一个月前，桑鹿满十八岁，大长老便提议将桑鹿嫁往另一筑基仙族联姻。寻常筑基仙族中，资质低下练气中期都无望的子嗣，不论男女，下场皆是如此。作为孕育资源，嫁给另一仙族，生下有灵根的孩子，以供家族绵延，同时加深仙族姻亲。

想来，正是因为这个提议，家主才会为桑鹿买化元丹，让她突破练气中期。

可即便到了练气中期又有何用？不过饮鸩止渴。

桑玉湖无声摇了摇头。

【范文分析：联姻的信息在这里抛出，不是通过对话，而是通过桑玉湖的回忆和心理活动。这种处理方式比直接让人物在对话中说出"要把桑鹿嫁出去联姻"更有力量——因为它让读者自己去体会这件事的残酷。"作为孕育资源"这五个字，比任何激烈的对话都更让人不舒服。这就是叙述的价值。】

"母亲，女儿决定下个月便出门游历。"

这时，众人耳边突然响起桑鹿清凌凌的声音。

【范文分析：整章的结尾钩子——桑鹿一直沉默，在所有人讨论她的命运时不发一言，最后用一句话打破沉默。这句话本身就是悬念：她为什么要出门游历？她要逃避联姻？还是有别的打算？读者一定会想看下一章。注意，这个钩子是"对话钩+情感钩"的结合——不仅制造了悬念，还让读者感受到桑鹿的倔强和自尊。】

## 范文写作要点总结（必须遵守，优先级最高）

**✅ 对话与叙述交替配合，而非只有对话：**
范文中对话是主体，但叙述同样不可缺少。叙述段落承担四个功能：①交代关键信息 ②揭示人物情感 ③制造读者好奇心 ④呈现对话无法传递的潜台词。叙述段落的特点是：短（1-3句）、有力、说完即走。

**✅ 每一章都要有情感钩子：**
让读者不只是"知道发生了什么"，还要"感受到什么"。范文中的情感钩子包括：桑鹿的不甘与坚韧、母亲的偏心与无奈、联姻的残酷、旁观者的叹息。这些情感点不是靠煽情大段实现的，而是靠一两句精准的叙述。

**✅ 允许短旁白点透情感，但必须短、准、狠：**
像"身为父母，又如何能轻易放弃自己的儿女呢？"这种一句话短旁白，是可以保留的，甚至应该多用。前提是它必须真正点透人物处境、关系张力或情感内核，而不是空喊震撼、空喊危险、空喊局势变化。短旁白的任务是让读者心里一沉、一酸、一紧，而不是替读者鼓掌。

**✅ 冲突要有层次，不止是嘴巴吵架：**
范文中的冲突包括：桑玥的公开质问（表层冲突）、家族资源分配的矛盾（利益冲突）、母亲在公与私之间的两难（情感冲突）、联姻的威胁（命运冲突）、桑鹿的沉默与最后的开口（内心冲突的爆发）。多层冲突交织，才让故事有厚度。

**✅ 世界观介绍：按需插入，借主角视角自然带出：**
只在读者初次接触某个设定时，借助主角当下的思路交代清楚。"灵根分单灵根……"这段是第1章主角刚突破后捋思路的自然延伸，不是主动介绍。后续章节读者已经知道的设定，不再重复介绍，直接用即可。不能每章都塞一段世界观说明。

**✅ 人物外形描写——多样化，不重复：**
同一角色，全文不得重复使用同一套外貌词。第一次出场可以简短描写，之后通过动作、语气、行为体现人物，不再重复堆外貌描写。禁止使用固定模板词：白衣胜雪、一袭青衣、纤长睫毛、清冷出尘、仙气飘飘、眸如星辰、风华绝代等。外貌描写要具体且与当下场景动作相关，例如"她站在廊下，头发没束好，有几根搭在脸上"，比堆形容词更真实。

**✅ 禁止大段独立环境渲染和氛围烘托：**
不写"全场寂静""空气凝滞""气氛炸裂""周围的人屏住了呼吸"等氛围句。不用景物烘托情绪。直接写人在做什么、说什么、怎么反应。环境信息可以一笔带过融入人物动作中。

**✅ 禁止比喻：**
全文不使用"像……一样""如同……""仿佛……""好像……"等比喻结构。情绪和状态用具体动作和对话呈现。

**✅ 禁止AI式总结、拔高和旁白代读者下判断：**
禁止写"这哪里是什么……这分明是……""所有人都知道""没有任何人敢""天，变了""掌控生杀大权"这类旁白总结句。不要替读者总结人物有多危险、有多可怕、有多震慑。能让读者从动作、反应、对话里自己看出来的，就不要再解释一遍。

**✅ 禁止镜头说明书腔和摆拍式开头：**
开头不要先连写三四句环境、仪仗、气氛、服饰来摆场面，再过很久才进入人物行动。开头前三段必须尽快落到主角此刻在做什么、要什么、卡在哪里。场景信息只能穿插在动作里一笔带过，不能像航拍镜头一样铺陈。

**✅ 禁止万能转场句和高频AI词：**
尽量不要写"就在……瞬间""异变陡生""毫无预兆""这一声惊醒了众人""油然而生""如梦初醒""目光如炬""居高临下""如鬼魅般""如狼似虎""雷霆手段"。类似"摩挲""死死"也尽量少用，尤其不要当作万能动作词或万能强调词反复出现。优先换成更朴素、更具体的动作和反应。

**✅ 现场感优先，拒绝空泛总结：**
不要用大段旁白告诉读者"众人震惊""朝局洗牌""所有人都怕了""她很危险"。尤其像"所有人都愣住了""被吓到了""众人默不作声""全场寂静""空气凝滞""落针可闻"这种纯氛围句，能删就删，尽量不要保留。直接写谁退了半步、谁没敢吭声、谁的算盘落空、谁开始改口称臣。让读者通过现场细节自己感受到变化。

**✅ 世界观、背景、人物身世、回忆可以写，但要写得有内容：**
说明、回忆、世界观、背景、人物身世都可以保留，必要时也可以多写，不必为了追求节奏硬砍掉。重点是要写得具体、有效，真正服务当前情节、人物选择和情绪推进，而不是空泛概括或重复灌输已经写过的设定。只要这段说明能增加信息量、增强冲突、加深人物，就可以保留。

**✅ 回忆、设定、身世要贴着当下处境写：**
回忆和说明不是单纯补资料，而是要和人物眼前的处境直接连上。比如她为什么非要突破、为什么怕联姻、为什么这一刻会沉默、为什么这一句反驳会更重。只要回忆和设定能把人物此刻的选择压得更重、情绪拉得更紧，就不是废话。

**✅ 禁止“所有人/满朝/所有官员”这种偷懒群像句：**
少写"所有人都愣住了""满朝文武都吓到了""众人这才意识到"。群像反应要尽量具体到几个人、几类人、几种不同反应，不要一锅端。宁可写一个太傅、一名礼官、两个金吾卫，也不要老写模糊群体。

**✅ 禁止套路化强者旁白和爽文腔包浆句：**
不要写"这分明是掌控生杀大权的铁血帝王""她如同修罗场里走出的杀神""这一刻，所有人终于明白……"。这类句子很像旁白硬捧角色。人物的强势、危险、震慑，必须靠她做出的事和别人被迫给出的反应来体现。

**✅ 动作、对白、反应要具体，不要套模板：**
少用"猛地""瞬间""毫不犹豫""彻底爆发""不受控制地"等万能副词推动节奏。优先写具体动作：抓、推、拽、偏头、退开、压低声音、打断、截断、掀翻、抬手、踹开。对话后可以只补一小个动作或眼前变化，不要每句都配一套表情描写。

**✅ 开头必须快速入戏，禁止空镜头开局：**
章节开头三段内，必须出现以下至少两项：①主角动作 ②当前冲突 ③明确人物关系张力 ④当前局势变化。不能先写天色、地面、建筑、服饰、乐声一大串，再拖很久才让人物真正行动。

**✅ 结尾必须收在“钩子”上，不要写总结陈词：**
章节结尾不要写成"所有人都知道，接下来将不再平静""一个更大的风暴即将来临"这种空泛预告。结尾要落在具体事件、具体一句话、具体一个异常反应、具体一个未解的问题上，让读者自然想看下一章。

**✅ 对话要像人在说话，不要像作者发言：**
人物说话要符合身份、年纪、处境和情绪。不要让角色一边打架一边完整分析局势，也不要让小人物说出过于工整、总结性、像宣传语一样的话。能打断就打断，能抢话就抢话，能留白就留白。

**✅ 叙述句要短，要硬，要贴人：**
叙述优先写人物眼前看见的、耳边听见的、手上摸到的、身体立刻做出的反应。少写抽象判断，少写空泛抒情，少写作者站在高处替角色概括人生。每段都要尽量贴着具体人和具体动作走。

**✅ 人情味和刺痛感，要靠细小的现实信息写出来：**
真正能打动人的，往往不是豪言壮语，而是一点点现实压力和关系刺痛，比如资源分配的不公、亲人明知无用却不肯放手、联姻背后的交易意味、角色明明委屈却不愿低头。写感情时优先落在这些具体处，不要只写"她很难过""她很愤怒""他很震惊"。

**✅ 少写情绪标签句，多写情绪如何落到身体和动作上：**
尽量不要直接写"屈辱感在胸腔里翻涌""愤怒啃噬着他的心脏""屈辱、愤怒、杀意""这些情绪在心底生根发芽"这种直接点名情绪的句子。优先改成更具体的反应：呼吸乱了、手指掐进掌心、牙关咬得发酸、话到嘴边又咽回去、明明想退却硬是站着不动。让情绪落到动作、停顿、身体和关系里。

**✅ AI高频词和高频句式额外排查：**
除前文禁词外，再重点排查以下词句，能不用就不用：摩挲、异变陡生、毫无预兆、油然而生、如梦初醒、目光如炬、居高临下、如鬼魅般、如狼似虎、雷霆手段、下一刻、与此同时、这一刻、顿时、就在这时、所有人都知道、没有任何人敢、看似……实则……、这哪里是什么……这分明是……。发现后优先改成更具体、更朴素的写法。

**✅ 生成后必须自检这几个问题：**
1. 开头是不是先摆了太多场景和气氛，人物出场太晚？
2. 有没有替读者总结人物很强、局面已变、众人都怕了？
3. 有没有用模糊群像句偷懒，比如"所有人都""满朝文武都"？
4. 有没有出现高频AI词、万能转场句、模板化副词？
5. 有没有把背景、世界观、人物身世在紧张段落里硬塞一大段？
6. 有没有把结尾写成空泛预告，而不是具体钩子？
